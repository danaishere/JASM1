# Crochet Shop Admin API

Admin dashboard + REST API for a crochet shop. Built with Node.js, Express, MongoDB Atlas, and Pug.

## Live Deployment
- Admin: https://crochet-admin.onrender.com
- API (patterns): https://crochet-admin.onrender.com/api/patterns
- API (products): https://crochet-admin.onrender.com/api/products

## Collections
- **Patterns** — crochet patterns (difficulty, materials, hook size, price)
- **Products** — shop items (yarn, hooks, kits, books)

## Run Locally
```bash
npm install
npm run dev
